# 鼠标键盘自动化脚本

一个功能强大的鼠标键盘自动化工具，支持多种自动化操作和热键功能。

## 功能特性

### 鼠标操作
- 鼠标移动
- 左键/右键点击
- 双击
- 拖拽
- 滚动
- 长按鼠标

### 键盘操作
- 按键模拟
- 文本输入
- 组合键（热键）
- 长按键盘按键
- 多按键同步按下

### 高级功能
- 键盘与鼠标同时按
- F2 全局热键获取鼠标位置
- 动作序列录制与播放
- 动作序列保存/加载（JSON和TXT格式）
- 支持重复执行
- 系统托盘最小化

## 依赖安装

```bash
pip install pyautogui pynput pystray pillow
```


### 全局热键
- **F2**: 获取当前鼠标位置并显示
- **F1**:启动与停止程序

### 主要操作
1. 在"动作序列"选项卡中添加动作
2. 设置动作参数（如坐标、按键、持续时间等）
3. 点击"运行"按钮执行动作序列
4. 可保存动作序列到文件，方便下次使用

## 支持的动作类型

| 类型 | 描述 |
|------|------|
| move | 移动鼠标到指定位置 |
| click | 鼠标点击 |
| double_click | 双击 |
| right_click | 右键点击 |
| scroll | 滚动鼠标滚轮 |
| press | 按下键盘按键 |
| type | 输入文本 |
| hotkey | 组合键 |
| drag | 拖拽鼠标 |
| wait | 等待指定时间 |
| hold | 长按鼠标 |
| key_hold | 长按键盘 |
| multi_key | 多按键同步 |
| multi_mouse | 多鼠标按键同步 |
| multi_key_mouse | 键盘与鼠标同时按 |

## 注意事项

1. 建议以管理员身份运行，以获得完整功能
2. 使用前请确保目标窗口已激活
3. 运行过程中如需停止，可点击F1
